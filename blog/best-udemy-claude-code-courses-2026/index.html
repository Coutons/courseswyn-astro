FAQ

What is Claude Code?
Anthropic’s CLI agentic coding tool. Reads your codebase, writes files, runs commands, executes tests, and manages development workflows autonomously.

What is Claude Cowork?
Anthropic’s desktop automation agent for non-developers. Automates files, emails, reports, and research through a visual interface.

Do I need coding skills?
No. Courses #1, #3, and #6 require zero coding background.

What is MCP?
Model Context Protocol — Anthropic’s open standard for connecting Claude to external tools, databases, and APIs.
