Course Content

About This Udemy Course

The following is the full official course description for Primavera P6 & Excel Integration with Power BI Dashboard as published on Udemy by instructor Akhilesh Shukla:

In today’s infrastructure projects, data-driven decision making is essential. Planning engineers and project control professionals must go beyond schedules and spreadsheets to create clear, professional dashboards for management and clients.

This course is designed specifically for infrastructure and construction professionals who want to transform Primavera P6 and Excel data into powerful Power BI dashboards.

You will learn how to:
  • Import and connect Primavera and Excel data into Power BI
  • Clean, structure, and transform raw project data using Power Query
  • Create data models and build relationships between multiple project datasets
  • Create different types of visualizations
  • Build S-Curves and Planned vs Actual analysis
  • Develop KPI dashboards for SPI, CPI, and progress monitoring
  • Perform cost and delay analysis using advanced visuals
  • Design interactive dashboards using slicers, filters, and drill-down features
  • Create dynamic reports suitable for client presentations and management reviews
  • Create a complete live project dashboard from start to finish
This course goes beyond theory and focuses on real-world application. You will work with practical infrastructure project data similar to metro rail, elevated corridor, and large-scale construction projects. Each concept is explained step-by-step so that even beginners can follow, while experienced professionals can enhance their reporting and analytical skills.

By the end of this course, you will confidently build professional, management-level dashboards that improve transparency, reporting efficiency, and project decision-making.
