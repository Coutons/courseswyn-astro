Course Content

About This Udemy Course

The following is the full official course description for Mastering Salesforce AI: Agentforce and Prompt Templates as published on Udemy by instructor Salesforce Developer:

Welcome to the Salesforce AI course, where we will learn to build Agents on Agentforce and prompt templates using Agent Builder and Prompt Builder to create AI-driven applications. We will learn to create simple custom actions as well as complex custom actions using Apex, flows, and REST API, will also understand how to boost Developers Productivity using Agentforce for Developers.

This course is designed to teach Agentforce Platform, How to Build Agents and all the different prompt templates from the very beginning to an advanced level. In this course, you will be able to learn every basic piece of information about Salesforce AI.

Agentforce Agents are AI assistant for Salesforce CRM which can be Deployed on multiple channels . It talks and understands like a human thanks to large language models such as OpenAI's GPT-4.

In this course, we will learn how to enable Agentforce, how to utilize the standard actions, and how to build custom actions which will be assigned to Agents to perform tasks that are specific to the business needs and can reduce lots of manual tasks that Salesforce Users have to perform on a daily basis.

  1. We will understand the architecture of Trusted AI + Data; we will delve into the Einstein Trust layer. We will learn to create Different types of custom actions using Apex, Flows, and Prompt templates.
  2. We will learn how to deploy these agents to a customer-facing site and how to transfer conversations from AI agents to human agents
  3. We will learn about data or record-triggered agents and how they can perform complex tasks in the background.
  4. We will learn about Agentforce and Data Cloud, explore the Agentforce Data Library and RAG, and understand how to work with unstructured data.
  5. We will learn about Agent Context Variables, AI versioning, and much more in Agent Builder.
  6. We will learn what a prompt template is and what a prompt builder is. Then, we will learn to create all four types of prompt templates: the sales email template, the field generation template, the record summary, and the most powerful, the Flex Template.
  7. We will also learn how this prompt template can invoke Apex, Flow, External API, and then we will learn how Apex, Flows, and Agent actions can invoke these prompt templates with grounded Salesforce data.
  8. We will learn about Einstein for Sales, Einstein for Service, and how these out-of-the-box Gen AI solutions can be leveraged.
  9. We Will Lean About Model Builder in Einstein Studio.
  10. We will Explore Agentforce for Developers and Compare it with the other AI Coding Assistant such as GitHub Copilot

  • Code related to each lecture.
  • New videos will be added in the coming future with other Salesforce AI topics.
