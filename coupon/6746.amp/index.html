About This Course

<div>Descubre cómo llevar tus aplicaciones de .NET al siguiente nivel integrando inteligencia artificial de forma práctica, moderna y lista para producción. En este curso aprenderás paso a paso cómo incorporar capacidades de IA dentro de tus aplicaciones, sin demasiada teoría y enfocado en escenarios reales.</div><div><br></div><div>Comenzaremos desde lo más básico, integrando IA en una aplicación de consola, para que entiendas los fundamentos de cómo interactuar con modelos de lenguaje. Luego avanzaremos hacia el uso de Microsoft Extensions AI, una de las piezas clave del ecosistema moderno de .NET para trabajar con IA de forma desacoplada y flexible, permitiéndote cambiar entre proveedores como OpenAI, Anthropic o Google sin reescribir tu aplicación.</div><div><br></div><div>A medida que progresamos, construiremos un chatbot en ASP.NET Core completamente funcional. Este no será un simple generador de texto: implementaremos streaming de respuestas para mejorar la experiencia del usuario, permitiendo que el texto se muestre en tiempo real sin esperas innecesarias. Además, aprenderás a utilizar tool calling, una técnica que permite a la IA ejecutar funciones de tu código C#, integrándose con tu base de datos o consumiendo APIs externas.</div><div><br></div><div>También abordaremos escenarios más avanzados, como el uso de human-in-the-loop, donde la IA solicitará aprobación antes de ejecutar acciones sensibles, como el envío de correos electrónicos. Esto es clave para construir sistemas seguros y confiables en entornos reales.</div><div><br></div><div>En el módulo de RAG (Retrieval-Augmented Generation), aprenderás a crear aplicaciones que permiten a la IA responder preguntas basadas en documentos propios de tu empresa. No solo eso, sino que implementaremos un sistema que cita las fuentes utilizadas, aumentando la confiabilidad de las respuestas y permitiendo validar la información.</div><div><br></div><div>Finalmente, exploraremos el Model Context Protocol (MCP), donde construiremos un servidor que permitirá que herramientas como ChatGPT o Claude interactúen directamente con tu aplicación. Esto abre la puerta a escenarios donde un usuario puede ejecutar acciones reales en tu sistema simplemente mediante lenguaje natural.</div><div><br></div><div>Este curso está diseñado para desarrolladores de .NET que quieren aprender haciendo, construyendo proyectos reales que pueden aplicar directamente en su trabajo o productos. No necesitas experiencia previa en inteligencia artificial, pero sí conocimientos básicos de C# y ASP.NET Core.</div><div><br></div><div>Al finalizar, no solo entenderás cómo funciona la IA moderna, sino que tendrás la capacidad de integrarla de forma efectiva en tus propias aplicaciones, creando soluciones más inteligentes, automatizadas y alineadas con las tendencias actuales de la industria.</div><div><br></div><div>Si quieres aprender a construir aplicaciones de .NET potenciadas por inteligencia artificial y agentes, este curso es para ti.</div>

What you'll learn:

  • Integrar IA en aplicaciones de .NET
  • Construir chatbots en ASP.NET Core
  • Hacer que la IA pueda llamar funciones de C# de tu app
  • Integrar IA con bases de datos y APIs externas
  • Implementar aprobación humana en acciones sensibles (que la IA pida permiso para hacer las cosas)
  • Crear un sistema RAG con documentos internos (que la IA responda preguntas en base a documentos)
  • Construir e integrar un servidor MCP
