What is Value Stream Mapping (VSM)?

Value Stream Map is a method in which you create a current state map of your business process, analyze it, and design the future state map. It is a visualization tool that has an ultra focus on identifying waste and eliminating it. Hence, it is also termed the heart of Lean Management.
